# Attribution

IslaGrid AI is built from public data. Every source below is shown here and surfaced in the UI footer.

## Grid operational data

- **LUMA Energy** — System Overview, BPS Monitoring, Planned Works, Service Updates. Public pages at `lumapr.com`.
- **Genera PR** — generation data published at `genera-pr.com`.
- **Puerto Rico Government Open Data** — `datos.pr.gov` generation-by-plant API.
- **AEE / PREPA** — load-shedding dashboard at `aeepr.maps.arcgis.com`.

## Regulatory and tariff data

- **Puerto Rico Energy Bureau (NEPR / PREB)** — `energia.pr.gov`. Approved rates and adjustment factors.

## Weather

- **National Weather Service / NOAA** — `api.weather.gov`. Public domain, US Government work.

## Geospatial

- **OpenStreetMap contributors** — power infrastructure layers via Overpass API. Data licensed under the [Open Database License (ODbL)](https://opendatacommons.org/licenses/odbl/). Map cartography style adapted from CARTO under [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/).
- **OpenInfraMap** — `openinframap.org`. Infrastructure visualization built on OSM data; same ODbL license.
- **U.S. Census Bureau — TIGER/Line** — Puerto Rico municipality (county subdivision) boundaries. Public domain.
- **CARTO Basemaps** — dark-matter raster tiles. Attribution: © OpenStreetMap contributors © CARTO.

## Solar

- **National Renewable Energy Laboratory (NREL)** — PV Rooftop Database for Puerto Rico, LiDAR collection 2015–2017. Published at `data.openei.org/submissions/2862`.
- **NREL PVWatts API v8** — solar production estimates. API used under the NREL Developer Network terms.

## Software

- Next.js, React, TypeScript, Tailwind CSS, MapLibre GL JS, Supabase, h3-js, pdfplumber, Playwright, httpx, pandas — see `package.json` and `ingestion/pyproject.toml` for full versions and licenses.
